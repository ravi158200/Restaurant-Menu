# RestoBite - Modern Restaurant Platform

## Setup Instructions

This repository contains the complete foundation for your modern restaurant web application.

### 1. Frontend Setup
Open a terminal in the `frontend` folder and run:
```bash
cd frontend
npm install
npm run dev
```

### 2. Backend Setup
Open a terminal in the `backend` folder and run:
```bash
cd backend
npm install
npm run dev
```

### 3. Generated Assets
I have generated beautiful UI assets (food photos) for the project. Check your Antigravity artifacts folder. To use them in the frontend, move them to `frontend/public/` with the names:
- `hero_food_bg.png`
- `pizza_menu.png`
- `pasta_menu.png`
- `burger.png` (rename one of them)

### Tech Stack Used
- Frontend: Vite, React, Tailwind CSS, Framer Motion, Lucide React
- Backend: Node.js, Express, Mongoose, Razorpay
- Features: Glassmorphism UI, Dark Mode, Admin APIs ready, Orders API ready.
